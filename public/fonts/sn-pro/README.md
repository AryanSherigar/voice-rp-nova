Place SN Pro webfont files in this folder for deterministic, self-hosted loading.

Expected filenames (woff2 preferred):
- SNPro-Regular.woff2
- SNPro-Medium.woff2
- SNPro-Bold.woff2

Optional legacy fallback filenames:
- SNPro-Regular.woff
- SNPro-Medium.woff
- SNPro-Bold.woff

When these files are missing, the app falls back to the system sans-serif stack defined in `index.css`.
